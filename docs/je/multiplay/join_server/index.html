进入服务器/局域网世界

要进入服务器，首先就要 获取服务器的IP地址，这里我们以进入Hypixel服务器为例子。

Hypixel服务器的IP地址是 mc.hypixel.net。复制这一串 IP地址，在游戏里打开 多人游戏 → 添加服务器。

在服务器地址那一栏填写或直接粘贴IP地址。（快捷键：Ctrl + C 复制，Ctrl + V 粘贴）

填写完后点击 完成。你就会发现有一个新添加的服务器。

如果服务器右上角的信号格是这样的，说明无法进入服务器或未能检测到服务器，需要检查服务器的拼写是否正确，若IP后有接端口号，请确保冒号为英文半角冒号 “:”，而不是中文全角冒号 “：”。

之后双击进入服务器就可以了。

当然Hypixel服务器是有 正版登陆 验证的，因此只有正版用户才能进去。
